目录构架
assets //资源的意思
里面包含 css 和 img

components //组件的意思
里面还有 common（支持所有移动端的公共组件） 和 content （针对当前项目所使用的公共组件）

router //路由相关的文件

store // vuex 状态管理工具

nextwork // 网络相关的封装文件

common //主要的 js 文件 里面的 const.js 主要是公共的常量 和公共的方法

views // 主要页面
